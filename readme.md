
# Jobee Api
Job management API, employers and employees job posting, geoposition search
and user token authentication.


## Usage/Examples

- npm install
- npm run dev

Go to http://localhost:3000/


### Tech Stack
- Node.js
- Express
- MongoDB
- Mongoose
- node-geocoder


## Deployment

https://my-job-api.herokuapp.com/
## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

- `PORT`
- `NODE_ENV`
- `DB_URI`
- `GEOCODER_PROVIDER`
- `GEOCODER_API_KEY`
- `JWT_SECRET`
- `JWT_EXPIRES_TIME`
- `COOKIE_EXPIRES_TIME`
- `SMTP_HOST`
- `SMTP_PORT`
- `SMTP_EMAIL`
- `SMTP_PASSWORD`
- `SMTP_FROM_NAME`
- `SMTP_FROM_EMAIL`
- `UPLOAD_PATH`
